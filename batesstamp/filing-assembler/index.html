About the Filing Assembler

What Is a Filing Assembler?

When filing a motion with exhibits in court, the standard practice is to assemble the documents in a specific order: the main motion or brief first, followed by each exhibit preceded by a labeled slip sheet (also called an exhibit cover page). This tool automates that assembly process, combining all of your documents into a single, court-ready PDF in seconds.

How It Works

Upload your main document (optional) and any number of exhibit files. The tool generates a slip sheet for each exhibit — a single page with the exhibit label centered in large text — then assembles everything in order: main document, then [Slip Sheet → Exhibit] for each exhibit. The result is a single PDF ready to e-file or print.

Drag-to-Reorder

Rearrange exhibits in any order by dragging the handle. Labels update automatically to reflect the new sequence.

Flexible Label Formats

Choose from Exhibit A/B/C, Exhibit 1/2/3, Plaintiff's Exhibit, Defendant's Exhibit, or a fully custom prefix.

Case Caption on Slip Sheets

Optionally include the court name, case name, and case number on each slip sheet for a professional, court-ready appearance.

Privacy First

All processing happens entirely in your browser. Your documents are never uploaded to any server — suitable for confidential and privileged materials.

Supported File Types

Both the main document and exhibit slots accept PDF, JPG, and PNG files. Image files are embedded as single-page PDFs in the output. For multi-page exhibits, use PDF format.

When to Use the Filing Assembler

  • Motion practice: Assemble a motion with supporting exhibits before e-filing
  • Trial preparation: Build a pre-marked exhibit binder as a single PDF
  • Deposition notices: Attach document requests as lettered exhibits
  • Discovery responses: Combine your response with labeled exhibit documents
  • Appellate filings: Assemble the appendix with tabbed exhibit pages
